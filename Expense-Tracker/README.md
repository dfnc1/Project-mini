# Expense Tracker CLI

---
A simple expense tracker application to manage your finances.
Example solution to the project [Expense Tracker](https://roadmap.sh/projects/expense-tracker) challenge from [roadmap.sh](https://roadmap.sh)

## Requirements
> - Python 3.10+

## Installation
```commandline
https://github.com/dfnc1/Project-mini.git
cd Expense-Tracker
```

## Usage
>```commandline
> python3 main.py add --desription "add expense" --amount 20 
>```
>```commandline
> python3 main.py list
>```
>```commandline
> python3 main.py summary 
>```
>```commandline
> python3 main.py summary --month 2
>```
>```commandline
> python3 main.py delete --id 2
>```
